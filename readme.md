# CuTranslater
既に中国語の字幕が入っている動画の中国語の字幕を日本語に変換し、中国語の字幕の上または下に表示します
字幕が動画に埋め込みの場合に使えます
概要程度であればわかるようになります

# 環境設定
Python 3.11が必要です
Python環境のセットアップがある程度できないと難しいと思います
install.batは他のPythonパッケージマネージャではで適宜読み替えてください
Windowsでしか動作確認していませんが、Windows依存の部分はほぼないので他の環境でも動くと思います
9から15行目はWindows依存なので他の環境の人はコメントアウトしてください

# 使い方
1. main.pyを起動します
2. 初回はモデルのロードがかなり長い時間かかります。これは最初の1回のみです
3. 動画を実際の大きさで表示し、"字幕エリアを選択"で中国語字幕の位置をおおまかに指定してください。この領域にOCRがかかります
4. "翻訳開始"を押して動画を再生してください。翻訳字幕は中国語字幕の上または下に出ます

注意点として、翻訳字幕を中国語字幕に重ねたいと思いますが、そうすると翻訳がおかしくなります
理由はウインドウは判別していないので、翻訳字幕自体をOCRにかけてしまうからです

# 使用しているライブラリ
PaddleOCR というBaiduのOCRを利用しています
最初はEasyOCRで実装しましたが精度が非常に悪く切り替えました
気になる人は注意してください
2.6.2とバージョンが古いのは3系はWindowsで正しく動かないためです
Python 3.11なのもPaddleOCRのバージョンに依存しています

翻訳はGoogleTranslatorです
_load_modelsの設定を変えれば他の言語のOCRや翻訳にも使えると思います
